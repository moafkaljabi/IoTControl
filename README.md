# IoT Backend

 IoT Backend implements a backend for IoT/Embedded devices using C++.and Golang.

 # C++ 
 C++ is used for the TCP server and client.


 # Golang 
 Golang is used for:
 - TCP client.
 - http server.
 - Database Communication; PostgresSQL.
 - JWT Authentication
